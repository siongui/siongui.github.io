全新安裝Ubuntu Linux注意事項

Updated: 八月 31, 2020
在Github上編輯

  • 下載 Ubuntu 20.04.1 LTS (Focal Fossa)

  • 檢查ISO檔 (HowToMD5SUM - Community Help Wiki):

    $ md5sum -c MD5SUMS

  • 開啟 Startup Disk Creator燒錄 (How to create a bootable USB stick on Ubuntu).

    • 會毀掉整個USB裝置裡的資料，並佔用2GB多的空間(佔用sdb1, sdb2)

      $ sudo fdisk /dev/sdb
$ sudo mkfs.ext4 /dev/sdb3

  • ubuntu fail to install grub lenovo - Google search, ubuntu fail to install grub - Google search

    • BIOS裡選legacy,不選UEFI
    • 安裝時問是否使用UEFI,選擇Go back(不要用UEFI)

  • 更新系統 [1]

    $ sudo apt-get update && sudo apt-get upgrade && sudo apt-get dist-upgrade

  • git autocomplete

    $ sudo apt-get install git bash-completion

  • 傳統中文 language pack

    $ sudo apt-get install language-pack-zh-hant

  • 工具(Vim, Chromium)

    $ sudo apt-get install vim
$ sudo apt-get install chromium-browser

  • Chromium extension and app

    • uBlock Origin
    • Google Translate
    • 新同文堂
    • LINE

  • ubuntu chinese input - Google search, [Ubuntu] 在 Ubuntu 18.04 中新增新酷音輸入法 | by MuLong PuYang | Medium

  • Lenovo筆電使用rtl8821ae無線網路斷線: 請看 台灣最佳CP值無作業系統(OS)筆記型電腦

  • 複製檔案數量多時，請看 [Bash] Copy Large Number of Files on Linux

  • 複製檔案時，注意權限問題:

    $ sudo cp -r src dst
$ sudo chown -R usr:grp *
