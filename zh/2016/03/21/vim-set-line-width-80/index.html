Vim編輯器設定寬度80的分隔線

March 21, 2016
在Github上編輯

Vim 編輯器設定寬度80的分隔線， 從 SO的回答 [1] ，在 .vimrc 裡加入：

if exists('+colorcolumn')
  set colorcolumn=80
else
  au BufWinEnter * let w:m2=matchadd('ErrorMsg', '\%>80v.\+', -1)
endif
