DuckDuckGo有趣線上小工具

九月 30, 2018
在Github上編輯

從駭客新聞 [1] 看到的資訊，發現 DuckDuckGo 跟 Google 一樣提供一些線上小工具/功能。這此把駭客新聞討論看到的有趣功能整理一下：

  • 漂亮地列印 JSON
  • 產生 QR Code
  • 解碼 URL
  • 解釋 crontab 命令
  • ASCII 藝術字
  • 貨幣匯率轉換
  • 產生 passphrase
  • 顏色選擇器, 2
  • 計算機
  • 產生隨機GUID
  • DuckDuckGo !Bang

非DuckDuckGo有趣連結：

  • GCHQ CyberChef
