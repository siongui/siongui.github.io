[CSS] Responsive Center Align Image

April 23, 2017
Edit on Github

CSS rule for HTML img element to be both responsive and centered horizontally.

CSS:

img.responsive-image-align-center {
    display: block;
    margin-left: auto;
    margin-right: auto;
    max-width: 100%;
    height: auto;
}
