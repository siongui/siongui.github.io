[AngularJS] Animate.css Test Demo

January 26, 2017
Edit on Github

I simplify the animations on GitHub Pages of Animate.css and convert it to AngularJS version.

Demo

We need only two directive to animate the text:

  • ngModel: bind select element to property named action, which record the
    value of selected option.
  • ngClass: combine the class animated and the value in action to
    animate the text in h1 element.
