[Bash] List Directory Size in Descending and Ascending Order

March 22, 2018
Edit on Github

Use du and sort command to list directory size in descending and ascending order.

List Folder Size (Descending)

$ du -s * | sort -rg

List Folder Size (Ascending)

$ du -s * | sort -g
