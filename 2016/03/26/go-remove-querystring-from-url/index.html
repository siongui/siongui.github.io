[Golang] Remove Query String From URL

March 26, 2016
Edit on Github

Remove query string from URL in Golang.

import "net/url"

func StripQueryString(inputUrl string) string {
      u, err := url.Parse(inputUrl)
      if err != nil {
              panic(err)
      }
      u.RawQuery = ""
      return u.String()
}

Tested on: Ubuntu Linux 15.10, Go 1.6.
