[Golang] Minify HTML

March 10, 2016
Edit on Github

Minify HTML via Golang.

The steps:

  1. Remove HTML comments [2]
  2. Remove all leading and trailing white space of each line.
  3. Pad a single space to the line if its length > 0.

Tested on: Ubuntu Linux 15.10, Go 1.6.
