Python try except Translated to Golang Synonyms

May 30, 2016
Edit on Github

Synonyms - Python try-except translated to Golang (Go programming language).

Python:

try:
    one()
    two()
    three()
except:
    handler()

Go:

func main() {
      defer func() {
              e := recover()
              if e != nil {
                      handler()
              }
      }()
      one()
      two()
      three()
}
